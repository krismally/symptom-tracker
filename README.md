# symptom-tracker
A CRUD web application to track symptoms for people with Chronic Illnesses. Based off of my own analog health tracker!

## Technologies Used
* HTML5
* CSS3
* JavaScript
* Mongoose
* MongoDB
* Node
* Express
* Bootstrap
* jQuery
